🌐 Translate Virtual Assistant

**Translate Virtual Assistant** është një platformë inteligjente që ndihmon përdoruesit në përkthime dhe komunikim të shpejtë duke përdorur teknologji të inteligjencës artificiale.

Sistemi përdor **OpenAI API (ChatGPT)** për të gjeneruar përkthime dhe përgjigje natyrale në kohë reale, duke e bërë procesin e përkthimit më të shpejtë dhe më efikas.

🚀 Përshkrimi i projektit

Ky projekt është një aplikacion web full-stack që kombinon backend dhe frontend modern për të ofruar një asistencë virtuale për përkthime dhe komunikim.

Përdoruesi mund të dërgojë tekst dhe sistemi kthen përkthim ose përgjigje të gjeneruar nga AI.

Teknologjitë e përdorura

### 🔹 Backend
- ASP.NET Core Web API (.NET)
- C#
- OpenAI API Integration

### 🔹 Frontend
- React.js
- TypeScript
- Axios (për komunikim me API)
- HTML & CSS
